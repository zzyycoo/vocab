# 词汇3500

高考英语核心词汇 · 乱序版

## 版本

| 版本 | 更新内容 |
|------|----------|
| v1.0.2 | 每次进入词汇表时重新随机排序 |
| v1.0.1 | 增加音标显示和发音功能（数据来源：Free Dictionary API） |
| v1.0.0 | 初始版本，词库分类（高频/中频/低频），闯关测试，错题复习，学习统计 |

## 功能

- **词汇书**：高频（800词）、中频（1350词）、低频（1350词）
- **闯关测试**：随机20题，4选1
- **错题复习**：查看历史错题
- **学习统计**：记录已学、测试次数和正确率
- **音标发音**：每个单词显示国际音标，可点击播放发音（需要网络）

## 技术说明

- 纯 HTML 单文件，无依赖，可离线浏览
- 进度保存至浏览器 localStorage
- 发音数据来自 [Free Dictionary API](https://dictionaryapi.dev/)，需要网络连接

## 访问

在线使用：https://zzyycoo.github.io/vocab/

本地使用：下载 `index.html` 直接在浏览器打开即可。

## 部署

将 `index.html` 上传至任意 GitHub 仓库的 `master` 分支根目录，启用 GitHub Pages 即可。
