## 功能

- **词汇书**：高频（800词）、中频（1350词）、低频（1350词）
- **闯关测试**：随机20题，4选1
- **错题复习**：查看历史错题
- **学习统计**：记录已学、测试次数和正确率
- **音标发音**：每个单词显示国际音标，可点击播放发音（Web Speech API 美式发音）

## 技术说明

- 纯 HTML 单文件，无依赖，可离线浏览
- 进度保存至浏览器 localStorage
- 发音使用 Web Speech API（美式英语，无需网络）

## 访问

将 `index.html` 上传至任意 GitHub 仓库的 `master` 分支根目录，启用 GitHub Pages 即可。
