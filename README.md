# 词汇3500

高考英语核心词汇 · 乱序版

## 功能

- **词汇书**：高频（800词）、中频（1350词）、低频（1350词）
- **闯关测试**：随机20题，4选1
- **错题复习**：查看历史错题
- **学习统计**：记录已学、测试次数和正确率

## 版本

- **v1.0.0** · 乱序版（每类别内部随机打乱）
- 纯 HTML 单文件，无依赖，可离线使用
- 进度保存至浏览器 localStorage

## 访问

在线使用：https://zzyycoo.github.io/vocab/

本地使用：下载 `index.html` 直接在浏览器打开即可。

## 部署

将 `index.html` 上传至任意 GitHub 仓库的 `master` 分支根目录，启用 GitHub Pages 即可。
